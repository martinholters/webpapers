Trapezoidal rule vs. implicit midpoint method

Martin Holters

Introduction

Consider an autonomous ordinary differetial equation (ODE)

x˙=f(x)(1)\tag{1}\dot{x} = f(x) x˙=f(x)(1)

where xxx is the state vector. For general fff, sovling analytically is usually impossible and we have to resort to numerical methods that seek to compute a sequence xn≈x(nT)x^n\approx x(nT)xn≈x(nT), where TTT is the time-step, here assumed constant for simplicity. Furthermore, we only consider the initial value problem where x(0)=x0x(0)=x_0x(0)=x0​ is given and use x0=x0x^0 = x_0x0=x0​.

There are many methods to achieve this goal of which two will be considered here: the trapezoidal rule and the implicit midpoint method. In particular, we shall review their similarity.

Reminder of the implicit midpoint rule

Just as a brief reminder, for an ODE x˙=f(x)\dot{x} = f(x)x˙=f(x), the implcit midpoint method is given by

xn+1=xn+Tf(12(xn+1+xn)).(2)\tag{2} x^{n+1} = x^n + Tf\big(\tfrac{1}{2}(x^{n+1} + x^n)\big). xn+1=xn+Tf(21​(xn+1+xn)).(2)

The trapezoidal rule revisited

The trapezoidal rule applied it to an ODE x˙=f(x)\dot{x} = f(x)x˙=f(x) is usually stated as

xn+1=xn+T2(f(xn+1)+f(xn)).(3)\tag{3} x^{n+1} = x^n + \tfrac{T}{2}\left(f(x^{n+1}) + f(x^n)\right). xn+1=xn+2T​(f(xn+1)+f(xn)).(3)

But for some applications, another form is more suitable which will be derived next. We start by rewriting (3) as

xn+1−T2f(xn+1)=xn+T2f(xn)=:xn+12.(4)\tag{4} x^{n+1} - \tfrac{T}{2}f(x^{n+1}) = x^n + \tfrac{T}{2}f(x^n) =: x^{n+\frac12}. xn+1−2T​f(xn+1)=xn+2T​f(xn)=:xn+21​.(4)

In passing, we note that this gives rise to an interesting interpretation of the trapezoidal rule: A forward-Euler step of length T2\frac{T}{2}2T​ to reach xn+12x^{n+\frac12}xn+21​ followed by a backward-Euler step also of length T2\frac{T}{2}2T​ to reach xn+1x^{n+1}xn+1. But here, we use it to rewrite the trapezoidal rule in xn+12x^{n+\frac12}xn+21​ and xn−12x^{n-\frac12}xn−21​, where the latter is, from the left-hand side of (4), given by

xn−12=xn−T2f(xn).(5)\tag{5} x^{n-\frac12} = x^{n} - \tfrac{T}{2}f(x^{n}). xn−21​=xn−2T​f(xn).(5)

Combining this with the right-hand side definition of (4) for xn+12x^{n+\frac12}xn+21​, we therefore get

xn+12+xn−12=2xnxn+12−xn−12=Tf(xn)(6)\tag{6}\begin{split} x^{n+\frac12} + x^{n-\frac12} &= 2x^n \\ x^{n+\frac12} - x^{n-\frac12} &= Tf(x^{n}) \end{split} xn+21​+xn−21​xn+21​−xn−21​​=2xn=Tf(xn)​(6)

or equivalently

xn=12(xn+12+xn−12)(7)\tag{7} x^n = \tfrac12 \left(x^{n+\frac12} + x^{n-\frac12}\right) xn=21​(xn+21​+xn−21​)(7)

and

f(xn)=1T(xn+12−xn−12).(8)\tag{8} f(x^{n}) = \tfrac{1}{T}\left(x^{n+\frac12} - x^{n-\frac12}\right). f(xn)=T1​(xn+21​−xn−21​).(8)

Inserting (7) in (8) we get the alternative trapezoidal rule formulation

1T(xn+12−xn−12)=f(12(xn+12+xn−12))(9)\tag{9} \tfrac{1}{T}\left(x^{n+\frac12} - x^{n-\frac12}\right) = f\left(\tfrac12 \left(x^{n+\frac12} + x^{n-\frac12}\right)\right) T1​(xn+21​−xn−21​)=f(21​(xn+21​+xn−21​))(9)

or

xn+12=xn−12+Tf(12(xn+12+xn−12)).(10)\tag{10} x^{n+\frac12} = x^{n-\frac12} + Tf\left(\tfrac12 \left(x^{n+\frac12} + x^{n-\frac12}\right)\right). xn+21​=xn−21​+Tf(21​(xn+21​+xn−21​)).(10)

This is very similar to the implicit midpoint rule, but with a half-sample time shift. Also note that from the sequence obtained from (10), the actual states have to be recovered with (7).

The implicit midpoint method revisited

Given that we could rewrite the trapezoidal rule such that it looked like the implicit midpoint rule time-shifted by 12\frac{1}{2}21​, it should be possbile to also go the other way. Indeed, letting

xn+12=12(xn+1+xn)(11)\tag{11}x^{n+\tfrac12} = \frac{1}{2}(x^{n+1}+x^{n}) xn+21​=21​(xn+1+xn)(11)

we can rewrite (2) as

xn+1=xn+Tf(xn+12)(12)\tag{12} x^{n+1} = x^n + Tf\big(x^{n+\tfrac12}\big) xn+1=xn+Tf(xn+21​)(12)

i.e.

Tf(xn+12)=xn+1−xn=2(xn+12−xn)(13)\tag{13} Tf\big(x^{n+\tfrac12}\big) = x^{n+1} - x^n = 2(x^{n+\tfrac12} - x^n) Tf(xn+21​)=xn+1−xn=2(xn+21​−xn)(13)

utilizing xn+1=2xn+12−xnx^{n+1}=2x^{n+\tfrac12}-x^{n}xn+1=2xn+21​−xn obtained from (11). Time-shifting (12) and utilizing xn−1=2xn−12−xnx^{n-1}=2x^{n-\tfrac12}-x^{n}xn−1=2xn−21​−xn from a time-shifted (11), we similarly get

Tf(xn−12)=(xn−xn−1)=2(xn−xn−12).(14)\tag{14} Tf\big(x^{n-\tfrac12}\big) = (x^{n} - x^{n-1}) = 2(x^{n} - x^{n-\tfrac12}). Tf(xn−21​)=(xn−xn−1)=2(xn−xn−21​).(14)

Summing (13) and (14) yields

Tf(xn+12)+Tf(xn−12)=2(xn+12−xn−12)(15)\tag{15} Tf\big(x^{n+\tfrac12}\big) + Tf\big(x^{n-\tfrac12}\big) = 2(x^{n+\tfrac12} - x^{n-\tfrac12}) Tf(xn+21​)+Tf(xn−21​)=2(xn+21​−xn−21​)(15)

which can be rewritten to

xn+12=xn−12+T2(f(xn+12)+f(xn−12)),(16)\tag{16} x^{n+\tfrac12} = x^{n-\tfrac12} + \frac{T}{2}\left(f\big(x^{n+\tfrac12}\big) + f\big(x^{n-\tfrac12}\big)\right), xn+21​=xn−21​+2T​(f(xn+21​)+f(xn−21​)),(16)

indeed resembling a time-shifted version of the trapezoidal rule. To recover xnx^nxn, we can solve (13) or (14) to

xn=xn+12−T2f(xn+12)(17)\tag{17} x^n = x^{n+\tfrac12} - \frac{T}{2}f\big(x^{n+\tfrac12}\big) xn=xn+21​−2T​f(xn+21​)(17)

or

xn=xn−12+T2f(xn−12),(18)\tag{18} x^{n} = x^{n-\tfrac12} + \frac{T}{2}f\big(x^{n-\tfrac12}\big), xn=xn−21​+2T​f(xn−21​),(18)

respectively.

Application to implicit ODEs

Now consider a differential equation

g(x,x˙)=0(27)\tag{27}g(x, \dot{x}) = 0 g(x,x˙)=0(27)

and assume there exists a function fff such that

g(x,f(x))=0,(28)\tag{28}g(x, f(x)) = 0, g(x,f(x))=0,(28)

i.e. we have the corresponsing ODE x˙=f(x)\dot{x}=f(x)x˙=f(x), but fff may not be available in closed form, so evaluating it may require expensive numerical solving. Of course, with this definition of fff, (28) should also hold for xnx^nxn, so that we have

g(xn,f(xn))=0.(29)\tag{29}g(x^n, f(x^n)) = 0. g(xn,f(xn))=0.(29)

Now we may substitute (7) and (8) and obtain

g(12(xn+12+xn−12),1T(xn+12−xn−12))=0(30)\tag{30}g\left(\tfrac12 \big(x^{n+\frac12} + x^{n-\frac12}\big), \tfrac{1}{T}\big(x^{n+\frac12} - x^{n-\frac12}\big)\right) = 0 g(21​(xn+21​+xn−21​),T1​(xn+21​−xn−21​))=0(30)

which can be used to produce a sequence xn+12x^{n+\frac12}xn+21​ by repeated numerical solution and then recover xnx^nxn with (7). We have thus applied the trapezoidel rule to (27) without requiring fff. In fact, this way we can apply the trapezoidal rule to differential-algebraic equations (DAEs), which are of the form (27) but where no solution function fff exists. However, the trapezoidal rule may then loose its convenient properties e.g. with regard to stability.

For the implicit midpoint rule, we can argue similarly starting from

g(xn+12,f(xn+12))=0(31)\tag{31}g(x^{n+\frac12}, f(x^{n+\frac12})) = 0 g(xn+21​,f(xn+21​))=0(31)

and inserting (11) and f(xn+12)=1T(xn+1−xn)f\big(x^{n+\tfrac12}\big) = \tfrac{1}{T}(x^{n+1} - x^n)f(xn+21​)=T1​(xn+1−xn) from (13) to obtain

g(12(xn+1+xn),1T(xn+1−xn))=0.(32)\tag{32}g\left(\tfrac{1}{2}(x^{n+1} + x^n), \tfrac{1}{T}(x^{n+1} - x^n)\right) = 0. g(21​(xn+1+xn),T1​(xn+1−xn))=0.(32)

Like for the trapezoidal rule, fff is not required explicitly, but if it does not exist, the usual properties of the implicit midpoint rule may no longer hold.
